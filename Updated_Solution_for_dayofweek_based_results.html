ANALYSIS:

  1. Pattern : Above residual pattern suggests the daywise distribution in number of flights. Meaning, we can see that number of flights on a particular day of week is not varying much.

  2. Starting from March'13, our model is predicting closely for number of flights on Saturdays (Black dots).

  3. Also, we can observe the seasonal pattern representing less number of flights on Fridays (Purple). While Mondays, Tuesdays, Wednesdays, Thurdays and Sundays have almost same number of flights.

  4. One can also observe few outliers in this dataset, specially at the Mid and End of year.
